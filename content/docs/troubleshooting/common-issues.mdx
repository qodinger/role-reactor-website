## Bot Not Responding
<Steps>
  <Step>
    **Check Bot Status**
    - Is the bot online?
    - Try `/ping` to test if bot responds
  </Step>
  <Step>
    **Verify Permissions**
    - Bot needs "Send Messages" and "Use Slash Commands"
    - Check channel-specific permissions
  </Step>
  <Step>
    **Check Role Position**
    - Bot's role must be above roles it manages
    - Go to Server Settings > Roles to check hierarchy
  </Step>
</Steps>

## Roles Not Being Assigned
<Callout type="warning">
**Most Common Cause:** Bot's role is positioned below the roles it's trying to assign
</Callout>
<Steps>
  <Step>
    **Fix Role Hierarchy**
    1. Go to Server Settings > Roles
    2. Drag the bot's role above all roles you want it to manage
    3. Test role assignment again
  </Step>
  <Step>
    **Check Role Names**
    - Make sure role names in the command match exactly
    - Role names are case-sensitive
  </Step>
  <Step>
    **Verify Bot Permissions**
    - Bot needs "Manage Roles" permission
  </Step>
</Steps>

## Reactions Missing
<Steps>
  <Step>
    **Use Unicode Emojis**
    - Custom server emojis might not work in all situations
    - Stick to standard Unicode emojis for best results
  </Step>
  <Step>
    **Check Bot Permissions**
    - Bot needs "Add Reactions" and "Manage Messages"
  </Step>
</Steps>
