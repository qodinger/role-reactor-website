## Parameters
<Cards>
  <Card title="message_id (required)" description="ID of the message to delete" />

## Step-by-Step Example
1. Use `/list-roles` to find the message ID you want to delete.<br />
2. Run the command:
    ```bash
    /delete-roles message_id:123456789012345678
    ```
3. The bot will remove the message and all role reactions.

## Troubleshooting
<Callout type="warning">
- **Message not found:** Double-check the message ID from `/list-roles`.<br />
- **Permission errors:** Ensure the bot can delete messages and manage roles.
</Callout>
