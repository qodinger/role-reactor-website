## Step-by-Step Example
1. Run the command:
    ```bash
    /list-roles
    ```
2. The bot will reply with a list of all role messages and their IDs.

## Troubleshooting
<Callout type="warning">
- **No messages found:** Make sure you have created role messages with `/setup-roles`.<br />
- **Permission errors:** Ensure the bot can read messages in the relevant channels.
</Callout>
