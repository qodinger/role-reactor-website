## Step-by-Step Example
1. Run the command:
    ```bash
    /list-temp-roles
    ```
2. The bot will reply with a list of users, roles, and expiration times.

## Troubleshooting
<Callout type="warning">
- **No temporary roles found:** Make sure roles were assigned with `/assign-temp-role`.<br />
- **User not found:** Double-check the username or mention.
</Callout>
