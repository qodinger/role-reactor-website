## Step-by-Step Example
1. Run the command:
    ```bash
    /help
    ```
2. The bot will reply with a list of commands and brief descriptions.<br />
3. For detailed help on a command:
    ```bash
    /help command:setup-roles
    ```

## Troubleshooting
<Callout type="warning">
- **No response:** Make sure the bot is online and has permission to send messages.<br />
- **Command not found:** Double-check the command name.
</Callout>
